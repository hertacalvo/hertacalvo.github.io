PYXEL - Single player game in Python

Single-player game with two game modes where protagonist is a pixel whose main goal is to avoid enemies.

The first game mode has different kinds of enemies (e.g. one type shoots lasers), and the second mode has tokens to be collected.

More information about Pyxel (as well as souce code) available at the github repo or the devpost page.
